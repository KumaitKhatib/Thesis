# MSc Thesis Repository

This repository contains the code and supporting materials for my MSc thesis:

Predicting Employment Based on Academic Performance Indicators for Syrian Graduates in Germany Using Machine Learning

The thesis investigates whether academic and experience-related indicators can be used to predict employment outcomes among Syrian graduates who completed Bachelor’s or Master’s degrees at German universities.
## Overview

The project applies machine learning methods to survey-based research on employment outcomes among Syrian graduates from German universities.

The repository includes the analysis notebook, preprocessing steps, model training workflow, evaluation code, and generated figures used for the thesis.



## Data Availability

The dataset used in this thesis is not publicly available due to privacy and ethical considerations. Although the survey did not collect direct personal identifiers, the data are connected to a specific and limited participant group.

Access to the dataset cannot be provided through this repository. Researchers or interested parties who need further information about the data should contact the Gisma University of Applied Sciences through the appropriate academic or administrative channels.
